# Hotel Room Management System

## Deskripsi Project

Hotel Room Management System adalah website sederhana berbasis **HTML dan CSS** yang dibuat untuk membantu pengelolaan data kamar hotel secara visual dan terstruktur.

Project ini menampilkan daftar kamar hotel beserta informasi penting seperti:

* Nomor kamar
* Tipe kamar (Standard, Deluxe, Suite)
* Harga per malam
* Lantai kamar
* Status kamar (Available, Occupied, Under Maintenance)

Selain itu, terdapat halaman detail kamar yang menampilkan:

* Fasilitas kamar
* Foto interior kamar
* Deskripsi view kamar (City View / Sea View)
* Riwayat pembersihan terakhir

Project ini dibuat sebagai tugas mata kuliah **Pemrograman Web**.
